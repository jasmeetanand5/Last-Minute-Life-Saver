# 🚀 Last Minute Life Saver

An AI-powered productivity assistant built with **Google AI Studio** and **Gemini API** to help users beat procrastination, prioritize work intelligently, and complete tasks before deadlines.

App link :https://last-minute-life-saver-67643593756.asia-southeast1.run.app

## 📖 Overview

Last Minute Life Saver is designed for students and professionals who struggle with managing multiple deadlines. Unlike traditional task managers, it acts as an AI execution assistant that analyzes workload, predicts deadline risks, intelligently prioritizes tasks, and guides users through structured focus sessions to maximize productivity.

## ✨ Key Features

* 🤖 AI-powered Task Analysis using Gemini
* 📊 Intelligent Task Queue
* 🧠 AI-generated Focus Plans
* 🎯 Hyper Focus Mode
* ⏱️ Guided Session Progression with Break Timers
* 🚨 Panic Mode for urgent deadlines
* 🆘 Smart Rescue Plan
* ⚠️ Workload Conflict Detection
* 📈 Deadline Risk Prediction
* 🔍 Search and Task Filtering
* 🗑️ Multi-task Selection & Delete
* 📝 Demo Workspace for first-time users

## 🚀 Running Locally

```bash
npm install
npm run dev
```

Create a `.env` file using `.env.example` and provide your Gemini API key.

## 📦 Deployment

The application is deployed on **Google Cloud Run** through **Google AI Studio**.
